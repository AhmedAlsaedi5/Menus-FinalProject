نظام إدارة المطاعم وقوائم الطعام<br>
مشروع ويب مطور باستخدام ASP.NET Core MVC يهدف إلى تسهيل إدارة المطاعم وقوائم الطعام، مع توفير تجربة سهلة للمستخدمين لاستعراض المطاعم، تصفح الأطباق، وإضافة التقييمات والمراجعات.<br>
فكرة المشروع<br>
يوفر النظام منصتين رئيسيتين:<br>
•	المستخدم: يستطيع استعراض المطاعم، تصفح قوائم الطعام، إضافة المطاعم إلى المفضلة، وتقييم المطاعم والأطباق. <br>
•	صاحب المطعم: يستطيع إدارة مطعمه بالكامل من خلال لوحة تحكم خاصة، تشمل إدارة بيانات المطعم، المواقع، والأطباق.<br> 
المميزات<br>
إدارة الحسابات<br>
•	تسجيل مستخدم جديد.<br> 
•	تسجيل الدخول والخروج. 
•	نظام صلاحيات باستخدام ASP.NET Core Identity. <br>
•	فصل الصلاحيات بين المستخدم وصاحب المطعم. <br>
إدارة المطاعم <br>
•	إنشاء مطعم جديد. <br> 
•	تعديل بيانات المطعم.  <br>
•	رفع صورة للمطعم.<br> 
•	إضافة أكثر من موقع للمطعم.  <br>
•	ربط المطعم بعدة تصنيفات.  <br>
إدارة قائمة الطعام <br>
•	إضافة الأصناف. <br> 
•	تعديل وحذف الأصناف. <br>
•	تصنيف الأطباق حسب الفئة. <br> 
•	تفعيل أو إيقاف توفر الطبق.  <br>
•	رفع صورة لكل طبق.  <br>
مميزات المستخدم <br>
•	استعراض جميع المطاعم. <br> 
•	البحث والتصفية حسب التصنيف. <br> 
•	عرض قائمة الطعام الخاصة بكل مطعم.  <br>
•	إضافة المطاعم إلى المفضلة. <br> 
•	تقييم المطاعم.  <br>
•	تقييم الأطباق. <br> 
•	عرض متوسط التقييم وعدد المراجعات. <br> 
•	حساب المسافة التقريبية بين المستخدم والمطعم. <br> 
دعم اللغات <br>
•	دعم اللغتين العربية والإنجليزية. <br> 
•	دعم اتجاهي الكتابة (RTL/LTR). <br> 
التقنيات المستخدمة <br> 
•	ASP.NET Core MVC <br>
•	Entity Framework Core <br>
•	SQL Server <br>
•	ASP.NET Core Identity <br>
•	LINQ <br>
•	Bootstrap 5 <br>
•	Razor Views <br>
•	HTML / CSS / JavaScript <br>
أهداف المشروع<br>
•	تطبيق مفهوم Role-Based Authorization. <br> 
•	استخدام Entity Framework Core لبناء العلاقات والاستعلامات. <br> 
•	تطبيق نمط MVC بصورة عملية. <br> 
•	استخدام ViewModel لتنظيم البيانات بين الـ Controller والـ View. <br> 
•	دعم تعدد اللغات (Localization). <br>
•	بناء واجهة استخدام متجاوبة وسهلة الاستخدام. <br>
